CTRL + l = Go to and select current site url
CTRL + r = Reload current page
CTRL + Shift + r = Realod current page, ignoring cached content
CTRL + n = New Window
CTRL + Shift + n = New window in anonymous mode
CTRL + h = History page
CTRL + j = Downloads page
CTRL + t = New tab
CTRL + w = Close tab
CTRL + Shift + w = Close the current window
CTRL + Shift + t = Reopen previous closed tab
CTRL + f = Search keyword
CTRL + g = Jump to the next match of your search
CTRL + Shift + g = Jump to the previous match of your search
CTRL + TAB = Change to the right tab
CTRL + Shift + TAB = Change to the left tab
CTRL + [0-9] = Change to the n tab in sequence(If more than 9, the 9 will go to the last tab)
Alt + Left Arrow = Go to the previous site in the current tab stack
Alt + Right Arrow = Go to the next site in the current tab stack
CTRL + Shift + j = Open/Close developer tools
CTRL + o = Open a file in chrome
CTRL + u = Display current page html
